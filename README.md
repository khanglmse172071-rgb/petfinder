# petfinder
PetFinder là ứng dụng kết nối cộng đồng yêu động vật nhằm tìm kiếm thú cưng thất lạc. Nền tảng tích hợp bản đồ định vị trực quan, bộ lọc thông minh giúp số hóa việc đăng tin lạc/nhặt được và nhận nuôi. Mục tiêu rút ngắn "thời gian vàng", tối ưu hóa quy trình cứu hộ tự động và lan tỏa thông điệp nhân văn, giúp các bé sớm trở về nhà
